VendeMKM

Aplicación de escritorio Windows · .NET 8 · ClickOnce

⬇ Instalar VendeMKM
Requisitos Abre este enlace desde Microsoft Edge (recomendado) o desde Chrome con la extensión ClickOnce instalada.
La app requiere Windows 10/11 y .NET 8 Runtime (el instalador lo descarga automáticamente si no está presente).

Actualizaciones automáticas

Una vez instalada, la app comprueba si hay nuevas versiones al arrancar y avisa automáticamente.

Requisitos

  • Windows 10 / 11 (64-bit)
  • .NET 8 Runtime (se instala automáticamente)
  • Cuenta en CardMarket
